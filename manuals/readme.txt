This is a simulation for virus infection spread on populations.

WINDOWS:

  -Run a binary excutable version, go to folder "bin\" and run "hw4.jar".

  or

  -Build and run:
    Goto scripts folder.
      To build the source code, run script "build.cmd"
      To run the built program, run script "run.cmd"
      To generate javadoc again, run script "javadoccompile.cmd"


MAC or LINUX:

  -Run a binary excutable version, go to folder "bin\" and run "hw4.jar".

  or

  -Build and run:
    Goto scripts folder.
      To build the source code, run script "build.sh"
      To run the built program, run script "run.sh"
      To generate javadoc again, run script "javadoccompile.sh"

UNIT TESTS:
  -Run script "unit_test.sh" on macOS or Linux.
  -Run script "unit_test.cmd" on windows.
